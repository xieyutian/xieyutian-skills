# 获取文件内容

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | repo |
| head_sha | string | 是 | name |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("raw.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/:owner/:repo/raw/:head_sha/:name", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
