# 搜索仓库

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/search/repositories`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| page | integer | 否 | 当前的页码 最大为 100 |
| per_page | integer | 否 | 每页的数量，最大为 20 |
| q | string | 是 | 搜索关键字 |
| sort | string | 否 | 排序字段，last_push_at(更新时间)、stars_count(收藏数)、forks_count(Fork 数)，默认为最佳匹配 |
| order | string | 否 | 排序顺序 (默认:desc) |
| owner | string | 否 | 仓库所属空间地址(组织或个人的地址path) |
| fork | string | 否 | 是否搜索含 fork 的仓库，如果显示,受白名单限制，默认：否 |
| language | string | 否 | 筛选指定语言的仓库 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/search/repositories", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
