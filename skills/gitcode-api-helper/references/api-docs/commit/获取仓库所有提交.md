# 获取仓库所有提交

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/repos/:owner/:repo/commits`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | 仓库所属空间地址（企业、组织或个人的地址path） |
| repo | string | 是 | 仓库路径(path) |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| sha | string | 否 | 提交起始的SHA值 |
| path | string | 否 | 包含该文件的提交 |
| author | string | 否 | 提交作者的邮箱或个人空间地址(username/login) |
| since | string | 否 | 提交的起始时间，时间格式（2024-11-08T16:25:44Z） |
| until | string | 否 | 提交的最后时间，时间格式为（2024-11-08T16:25:44Z） |
| page | string | 否 | 当前的页码 |
| per_page | string | 否 | 每页的数量，最大为 100，默认 20 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/repos/:owner/:repo/commits", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
