# 仓库的某个提交

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/repos/:owner/:repo/commits/:sha`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | 仓库所属空间地址(组织或个人的地址path) |
| repo | string | 是 | 仓库路径(path) |
| sha | string | 是 | 提交的SHA值或者分支名 |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| show_diff | boolean | 否 | 默认为 false; 为 true 时，返回files字段, files字段中包含本地提交中的至多100个变更文件名 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/repos/:owner/:repo/commits/:sha", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
