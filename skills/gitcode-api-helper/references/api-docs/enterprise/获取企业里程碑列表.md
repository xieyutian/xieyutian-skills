# 获取企业里程碑列表

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/milestones`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | string | 是 | 企业id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| name | string | 否 | 名称 |
| state | string | 否 | 状态 |
| order_by | string | 否 | 排序字段。可选：updated_at |
| sort | string | 否 | 排序方式。降序：desc；升序：asc |
| page | string | 否 | 当前的页码:默认为 1 |
| per_page | string | 否 | 每页的数量，最大为 100，默认 20 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v8/enterprises/:enterprise/milestones", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
