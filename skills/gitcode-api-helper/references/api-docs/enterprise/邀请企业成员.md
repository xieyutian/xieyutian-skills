# 邀请企业成员

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/memberships/:username`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | integer | 是 | 企业id |
| username | string | 是 | 用户名(username/login) |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| permission | string | 否 | 成员权限: 拉代码(pull)，推代码(push)，维护者(admin)，customized(自定义角色)。默认: push |
| role_id | string | 否 | 角色ID, 如果permission传入了customized则需要传入角色id |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "permission": "string",

  "role_id": "string"

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v8/enterprises/:enterprise/memberships/:username", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
