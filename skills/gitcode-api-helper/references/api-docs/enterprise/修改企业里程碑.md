# 修改企业里程碑

## API 端点

**方法:** `PUT`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/milestones/:milestone_id`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | string | 是 | 企业id |
| milestone_id | string | 是 | 里程碑id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| title | string | 是 | 名称 |
| description | string | 否 | 描述 |
| start_date | string | 否 | 开始时间, 格式%Y-%m-%d |
| due_date | string | 否 | 截止时间, 格式%Y-%m-%d |
| state_event | string | 否 | 里程碑状态。开启：active；关闭：closed |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "title": "string",

  "description": "string",

  "start_date": "string",

  "due_date": "string",

  "state_event": "string",

  "projects": [

    0

  ]

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("PUT", "/api/v8/enterprises/:enterprise/milestones/:milestone_id", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
