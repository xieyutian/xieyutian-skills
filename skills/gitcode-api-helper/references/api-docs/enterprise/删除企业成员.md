# 删除企业成员

## API 端点

**方法:** `DELETE`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/members/:username`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | integer | 是 | 企业id |
| username | string | 是 |  |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("DELETE", "/api/v8/enterprises/:enterprise/members/:username", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
