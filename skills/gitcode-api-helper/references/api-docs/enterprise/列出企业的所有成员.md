# 列出企业的所有成员

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/members`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | integer | 是 | 企业id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| page | integer | 否 | 当前的页码:默认为 1 |
| per_page | integer | 否 | 每页的数量，最大为 100，默认 20 |
| role | string | 否 | 根据角色筛选成员(all/admin/member) |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v8/enterprises/:enterprise/members", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
