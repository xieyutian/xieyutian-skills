# 获取企业自定义角色

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v8/enterprise/:enterprise_id/customized_roles`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise_id | string | 是 | 企业id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 否 | 用户授权码 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v8/enterprise/:enterprise_id/customized_roles", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
