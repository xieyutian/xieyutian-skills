# 获取企业里程碑可以关联的项目列表

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/groups/projects`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | string | 是 | 企业id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| group_name | string | 否 | 组织名称 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v8/enterprises/:enterprise/groups/projects", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
