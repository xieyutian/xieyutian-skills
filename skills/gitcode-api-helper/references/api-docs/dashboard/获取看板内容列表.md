# 获取看板内容列表

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/org/:owner/kanban/:kanban_id/item_list`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | 组织的路径 |
| kanban_id | string | 是 | 看板id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 否 | 用户授权码 |
| repo | string | 否 | 仓库路径 |
| source_type | string | 否 | 来源类型：issue/merge_request |
| source_status | string | 否 | 来源状态：opened/closed/merged |
| title | string | 否 | 标题 |
| source_iids | string | 否 | 使用pr或者issue的iid进行查询, 使用该参数必须填写repo参数，否则不生效 |
| page | string | 否 | 分页参数，第几页 |
| per_page | string | 否 | 分页参数，每页数量 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/org/:owner/kanban/:kanban_id/item_list", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
