# 获取组织看板列表

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/org/:owner/kanban/list`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | 组织的路径 |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| status | integer | 否 | 状态 0: 正常 1:关闭，默认不传为全部 |
| sort | string | 否 | 最新: newest, 最老: oldest |
| visibility | integer | 否 | 可见性 1:公开的 2: 私密的 |
| search | string | 否 | 按照名称搜索 |
| page | string | 否 | 当前页码 |
| per_page | string | 否 | 每页数量 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/org/:owner/kanban/list", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
