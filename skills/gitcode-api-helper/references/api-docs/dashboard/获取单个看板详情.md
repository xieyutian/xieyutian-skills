# 获取单个看板详情

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/org/:owner/kanban/:kanban_id/detail`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | 组织的路径 |
| kanban_id | string | 是 | 看板id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 否 | 用户授权码 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/org/:owner/kanban/:kanban_id/detail", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
