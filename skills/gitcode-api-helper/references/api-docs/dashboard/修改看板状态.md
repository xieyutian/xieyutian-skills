# 修改看板状态

## API 端点

**方法:** `PUT`
**端点:** `https://api.gitcode.com/api/v5/org/:owner/kanban/:kanban_id/state`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | 组织的路径 |
| kanban_id | string | 是 | 看板id |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 否 | 用户授权码 |
| state | string | 是 | 看板状态，关闭：closed；开启：open |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "state": "string"

})

headers = {

  'Content-Type': 'application/json'

}

conn.request("PUT", "/api/v5/org/:owner/kanban/:kanban_id/state", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
