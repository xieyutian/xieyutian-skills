# 列出授权用户所属的组织

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/users/orgs`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| page | string | 否 | 当前的页码:默认为 1 |
| per_page | string | 否 | 每页的数量，默认为20，最大为100 |
| admin | boolean | 否 | 筛选有管理员权限 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/users/orgs", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
