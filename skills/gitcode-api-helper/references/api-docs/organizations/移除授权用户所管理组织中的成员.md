# 移除授权用户所管理组织中的成员

## API 端点

**方法:** `DELETE`
**端点:** `https://api.gitcode.com/api/v5/orgs/:org/memberships/:username`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| org | string | 是 | 组织的路径(path/login) |
| username | string | 是 | 用户名(username/login) |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("DELETE", "/api/v5/orgs/:org/memberships/:username", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
