# 更新授权用户的资料

## API 端点

**方法:** `PATCH`
**端点:** `https://api.gitcode.com/api/v5/user`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |

### HEADER PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| avatar | string | 否 | 头像 |
| nickname | string | 否 | 用户昵称 |
| company | string | 否 | 公司 |
| description | string | 否 | 个人简介 |
| email | string | 否 | 个人邮箱（需要通过认证处理） |
| github_account | string | 否 | Github账号 |
| website | string | 否 | 个人网站 |
| location | string | 否 | 所在地 |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "avatar": "string",

  "nickname": "string",

  "company": "string",

  "description": "string",

  "email": "string",

  "github_account": "string",

  "website": "string",

  "location": "string"

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("PATCH", "/api/v5/user", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
