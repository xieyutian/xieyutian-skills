# 标记一个仓库里的通知为已读

## API 端点

**方法:** `PUT`
**端点:** `https://api.gitcode.com/api/v5/repos/:owner/:repo/notifications`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | repo |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| ids | string | 否 | 指定一组通知 ID，以 , 分隔 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': '*/*'

}

conn.request("PUT", "/api/v5/repos/:owner/:repo/notifications", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
