# 创建个人项目仓库

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/user/repos`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| name | string | 是 | 仓库名称 |
| description | string | 否 | 仓库描述 |
| has_issues | boolean | 否 | 允许提Issue与否。默认: 允许(true) |
| has_wiki | boolean | 否 | 提供Wiki与否。默认: 提供(true) |
| can_comment | boolean | 否 | 允许用户对仓库进行评论。默认：允许(true) |
| auto_init | boolean | 否 | 值为true时则会用README初始化仓库。默认: 不初始化(false) |
| gitignore_template | string | 否 | Git Ignore模版 |
| license_template | string | 否 | License模版 |
| path | string | 否 | 仓库路径 |
| private | boolean | 否 | 是否私有 |
| default_branch | string | 否 | 初始化项目时的默认分支名称。默认: main |
| import_url | string | 否 | 导入项目的git地址，import_url和project_template参数职能传入一个 |
| project_template | string | 否 | 模板项目的path，import_url和project_template参数职能传入一个 |
| repository_type | string | 否 | 仓库类型。code:代码仓，model:模型仓，dataset:数据集仓，space:space，默认code |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "name": "string",

  "description": "string",

  "has_issues": True,

  "has_wiki": True,

  "can_comment": True,

  "auto_init": True,

  "gitignore_template": "string",

  "license_template": "string",

  "path": "string",

  "private": True,

  "default_branch": "string",

  "import_url": "string",

  "project_template": "string",

  "repository_type": "string"

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/user/repos", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
