# 列出一个仓库里的通知

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/repos/:owner/:repo/notifications`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| owner | string | 是 | repo |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| unread | boolean | 否 | 是否未读，默认true |
| type | string | 否 | 筛选指定类型的通知，all：所有，event：事件通知，referer：@ 通知 |
| since | string | 否 | 只获取在给定时间前更新后的消息，要求时间格式为 ISO 860 |
| before | string | 否 | 只获取在给定时间前更新的消息，要求时间格式为 ISO 860 |
| ids | string | 否 | 指定一组通知 ID，以 , 分隔 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/repos/:owner/:repo/notifications", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
