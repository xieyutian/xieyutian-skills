# 列出授权用户所有仓库

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/user/repos`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| visibility | string | 否 | 公开(public)、私有(private)或者所有(all)，默认: 所有(all) |
| affiliation | string | 否 | owner(授权用户拥有的仓库)、collaborator(授权用户为仓库成员)、organization_member(授权用户为仓库所在组织并有访问仓库权限)、enterprise_member(授权用户所在企业并有访问仓库权限)、admin(所有有权限的，包括所管理的组织中所有仓库、所管理的企业的所有仓库)。可以用逗号分隔符组合。如: owner, organization_member 或 owner, collaborator, organization_member |
| type | string | 否 | 筛选用户仓库: 其创建(owner)、个人(personal)、其为成员(member)、公开(public)、私有(private)，不能与 affiliation 参数一并使用，否则会报 422 错误，与visibility参数一起使用，visibility参数拥有更高的优先级 |
| sort | string | 否 | 排序方式: 创建时间(created)，更新时间(updated)，最后推送时间(pushed)，仓库所属与名称(full_name)。默认: full_name |
| direction | string | 否 | 如果sort参数为full_name，用升序(asc)。否则降序(desc) |
| q | string | 否 | 搜索关键字 |
| page | integer | 否 | 当前的页码 |
| per_page | integer | 否 | 每页的数量，最大为 100，默认 20 |
| repo_type | string | 否 | 筛选仓库类型，可以是 code：代码仓，model：模型仓，dataset：数据集仓，space ：space，all：所有仓库类型。默认: code |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/user/repos", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
