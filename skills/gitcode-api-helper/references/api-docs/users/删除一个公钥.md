# 删除一个公钥

## API 端点

**方法:** `DELETE`
**端点:** `https://api.gitcode.com/api/v5/user/keys/:id`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| id | string | 是 | 公钥 ID |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("DELETE", "/api/v5/user/keys/:id", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
