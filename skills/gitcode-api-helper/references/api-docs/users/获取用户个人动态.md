# 获取用户个人动态

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v5/users/:username/events`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| username | string | 是 |  |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| year | string | 否 | 开始年（2024） |
| next | string | 否 | 结束日期 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v5/users/:username/events", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
