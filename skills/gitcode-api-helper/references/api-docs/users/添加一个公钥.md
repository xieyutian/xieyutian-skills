# 添加一个公钥

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/user/keys`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| key | string | 是 | 公钥内容 |
| title | string | 是 | 公钥名称 |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "key": "string",

  "title": "string"

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/user/keys", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
