# 转移仓

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/org/:org/projects/:repo/transfer`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| org | string | 是 | 仓库所属组织 |
| repo | string | 是 | 仓库路径 |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| transfer_to | string | 是 | 要转移到的目标组织 |
| password | string | 是 | 用户密码 |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "transfer_to": "string",

  "password": "string"

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/org/:org/projects/:repo/transfer", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
