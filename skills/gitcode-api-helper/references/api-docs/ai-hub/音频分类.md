# 音频分类

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/audio/classification`

## Request

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| model | string | 是 | 模型名称，默认为 audeering/wav2vec2-large-robust-12-ft-emotion-msp-dim |

### HEADER PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Content-Type': 'multipart/form-data',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/audio/classification", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
