# 图像生成视频（查询状态）

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/video/status`

## Request

### HEADER PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| request_id | string | 是 | 请求ID |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "request_id": "009522a671394f3f5ac02c63b1a62662"

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/video/status", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
