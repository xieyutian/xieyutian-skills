# 图像生成视频（创建任务）

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/video/generate`

## Request

### HEADER PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| img_url | string | 是 | 支持输入 Base64 或图片 URL |
| model | string | 是 | 模型名称 |
| prompt | string | 是 | 提示词 |


## 代码示例

### Python

```python
import http.client

import json



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = json.dumps({

  "img_url": "https://cdn-static.gitcode.com/_nuxtaihub/chatexample_67.png",

  "model": "Wan2.2-I2V-A14B",

  "prompt": "小猫在散步",

  "seed": 0

})

headers = {

  'Content-Type': 'application/json',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/video/generate", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
