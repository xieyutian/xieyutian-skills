# 自动语音识别

## API 端点

**方法:** `POST`
**端点:** `https://api.gitcode.com/api/v5/audio/transcriptions`

## Request

### HEADER PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| model | string | 是 | 模型名称，默认为 openai/whisper-large-v3 |
| temperature | string | 否 | 采样温度（0.0-1.0），默认为0.6 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Content-Type': 'multipart/form-data',

  'Accept': 'application/json'

}

conn.request("POST", "/api/v5/audio/transcriptions", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
