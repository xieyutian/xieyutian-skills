# 获取标签列表

## API 端点

**方法:** `GET`
**端点:** `https://api.gitcode.com/api/v8/enterprises/:enterprise/labels`

## Request

### PATH PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| enterprise | string | 是 | 仓库所属空间地址（企业、组织或个人的地址path） |

### QUERY PARAMETERS

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| access_token | string | 是 | 用户授权码 |
| search | string | 否 | 关键词查询 |
| direction | string | 否 | 排序方向 asc/desc |
| page | string | 否 | 当前的页码 |
| per_page | string | 否 | 每页的数量，最大为 100，默认 20 |


## 代码示例

### Python

```python
import http.client



conn = http.client.HTTPSConnection("api.gitcode.com")

payload = ''

headers = {

  'Accept': 'application/json'

}

conn.request("GET", "/api/v8/enterprises/:enterprise/labels", payload, headers)

res = conn.getresponse()

data = res.read()

print(data.decode("utf-8"))
```
